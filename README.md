1. AdapterDP — Adapter Pattern
Var olan bir arayüzü (Onceki) yeni bir arayüze uyarlamayı gösterir (Adapter/Adaptee/Target).

2. DecoratorDP — Decorator Pattern
Nesnenin davranışını runtime’da sarmalayarak genişletme örneği.

3. FactoryDP — Factory Pattern (Simple/Factory Method örneği)
IPizza / PizzaDukkani yapısıyla nesne oluşturmayı merkezileştirme (pizza-factory).

4. FactoryDP2 — Factory Pattern (muhtemelen gelişmiş varyant)
FactoryDP’nin başka bir versiyonu; daha ileri factory varyasyonu veya refactor örneği.

5. OCPExmp1 — SOLID: Open/Closed Principle (prensip örneği)
Açık/Kapalı prensibini gösteren örnek uygulama.

6. OCPExmp2 — SOLID: Open/Closed Principle (diğer örnek)
OCP’yi farklı bir yaklaşımla veya genişletme örneğiyle gösteren versiyon.

7. StrategyDP — Strategy Pattern
Bir algoritmayı soyutlayıp runtime’da farklı stratejilerle değiştirme örneği.

8. Uyg1 — Temel OOP örneği (sınıflar/metotlar)
OOP temellerini (sınıf, metot, nesne oluşturma) gösteren basit uygulama.

9. Uyg1Abstract — Abstract Class / Abstraction örneği
Abstract sınıf ve abstract metot kullanımıyla soyutlama örneği.

10. Uyg1Constructor — Constructor (yapıcı metot) örneği
Farklı constructor kullanımları, nesne inisyalizasyonu.

11. Uyg1Odev — Uygulama/Ödev (muhtemelen birleşik OOP kavramları)
Bir ödev uygulaması; birden çok OOP kavramını birleştiriyor olabilir.

12. Uyg1OperatorOverloading — Operator Overloading (operatör aşırı yükleme)
C# operatör aşırı yükleme örnekleri (operator +, -, vb.).

13. Uyg2Generic — Generics (Generic tipler/methods) örneği
Tip güvenli ve yeniden kullanılabilir generic sınıf/metot kullanımı.

14. Uyg2Generiic — Generics
Yine Generics konusuna ait.

15. Uyg2SealedMethod — sealed (override engelleme) örneği
sealed anahtar kelimesi ile metod veya sınıf kapatma kullanımını gösterir.

16. Uyg2Singleton — Singleton Pattern
Tekil instans sağlama örneği

17. Uyg3IntsizAbs — Interface ve Abstract
interface/abstract kullanım farklarını gösteren örnek (soyutlama karşılaştırması).

18. Uyg3Kalitim — Inheritance (Kalıtım) örneği
Base/derived sınıflarla miras ve override kullanımını gösterir.

19. Uyg3Property — Properties (get/set) örneği
C# property (encapsulation için) örnekleri; validation/getter/setter kullanımı.

20. Uyg4Donusum — Type Conversion / Casting örneği
implicit/explicit dönüşümler veya tip dönüşümü örnekleri.

21. Uyg4IntliMulti — Interface’li Çoklu (Multiple interface implementation) örneği
Bir sınıfın birden fazla interface’i implement etmesi ile çoklu davranış gösterimi.

22. Uyg4Sekil — Polymorphism / Shape example (Strategy/Factory ile beraber de olabilir)
Shape (Sekil) örneği: kalıtım ve polymorphism (alan/çevre hesaplama vb.) tipik örnek.

23. Uyg5ArraySrt — Array işlemleri + Sorting 
Dizi/algoritma uygulaması (sıralama)
